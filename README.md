# My Magical Day ✨

A kawaii pixel-art daily planner inspired by the original Digital Typewriter / Receipt Printer concept — rebuilt with a cuter design, live editing, and persistent data.

## Features

- 🖨️ **Animated pixel printer** — prints your day as a receipt with a heart-sliding animation and a pop-in reveal effect
- 📝 **Editable events & to-do quests** — add or delete items through a bow-ribbon pixel modal; events have time + category, quests have categories
- 🐰 **Mood charm** — pick from Sparkly 🐰, Cozy ☕, Brave 🌈, or Sleepy 🌙; changes the avatar, affirmation text, and receipt feel
- 🎀 **Receipt themes** — choose Strawberry, Lavender, or Mint to recolor the receipt's border and accents
- 💾 **LocalStorage persistence** — your name, mood, theme, events, and quests are saved and restored on refresh
- ✅ **Click to complete** — click a quest to mark it done with a sparkle burst; click again to undo
- 📊 **Live progress stats** — event count, quest count, and a pixel gradient progress bar update in real time
- 🌸 **Pastel pixel aesthetic** — floating pixel decorations, background sparkles, bobbing title, and wobbling avatar; Sanrio × Stardew × lo-fi inspired

Open `my-magical-day.html` in a browser — no server or install needed.
